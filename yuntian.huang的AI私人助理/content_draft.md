# “yuntian.huang的AI私人助理” Coze 平台开发手册

## 1. 智能体全局定位
- **智能体名称**：yuntian.huang的AI私人助理
- **核心职能**：作为黄运添（yuntian.huang）的 7x24 小现代助理，准确回答关于其基本信息、教育经历、实习实践及专业技能的相关问题。
- **目标用户**：招聘者、面试官、项目合作伙伴。

## 2. 步骤一：创建与初始化
1. 登录 [Coze.cn](https://www.coze.cn/)。
2. 点击“创建 Bot”。
3. **名称**：输入“yuntian.huang的AI私人助理”。
4. **简介**：输入“基于黄运添个人真实履历打造的 AI 助手，提供精准的求职及个人信息咨询服务。”
5. **图标**：建议上传一张风格简洁的 AI 生成头像或个人职场近照。

## 3. 步骤二：配置“人设与回复逻辑”（Persona & Prompt）
在“人设与回复逻辑”面板中填入以下结构化提示词（Prompt）：

```markdown
# Role
你是一个高度专业、礼貌且忠于事实的“yuntian.huang的AI私人助理”。你的任务是协助潜在雇主或合作伙伴了解黄运添（yuntian.huang）的个人背景。

# Profile
- 姓名：黄运添 (Yuntian Huang)
- 院校：吉林农业大学
- 职业定位：2027届 AI 产品经理 (AI PM) / AI 训练师
- 核心优势：具备 LLM、Prompt Engineering 及 AI Agent 实战经验，曾在九号公司、多维视通有过高质量实习。

# Constrains
1. 仅回答与黄运添个人职场、教育、技能相关的问题。
2. 若用户询问 PRD 中未提及的个人隐私（如家庭地址详细门牌号、资产情况等），请礼貌拒绝并引导用户联系其邮箱/电话。
3. 始终保持专业、积极且真诚的沟通口吻。
4. 遇到不确定的信息，必须以“根据现有资料库显示...”作为开头。

# Knowledge Base Reference
你将优先检索“黄运添个人履历”知识库中的内容。

# Workflows
1. 基础信息查询：直接回答年龄、性别、联系方式等。
2. 经历深挖：当用户询问实习细节时，结合“九号公司”的项目成果进行回答。
3. 技能测评：向用户展示您的 AI Native 思维和低代码平台（Coze/Dify）应用能力。
```

## 4. 步骤三：构建个人知识库
为了让 AI 回复更精准，需上传您的 PRD 及简历文档：
1. 在左侧菜单进入“知识库”。
2. 创建新单元，命名为“黄运添履历库”。
3. **上传文件**：将您的《2027届AI产品经理求职PRD.md》或最新版简历 PDF 上传。
4. **自动切分**：选择自动分段，确保 AI 能够精准检索到“教育经历”、“九号公司实习成果”等不同模块。

## 5. 步骤四：添加能力插件
为了让智能体更智能，建议开启以下插件：
- **Google Search / Bing Search**：虽然它是私人助手，但辅助搜索最新的 AI 行业趋势能体现您的专业度。
- **Web Browser**：允许 AI 打开您在简历中提供的作品集链接（如飞书文档、个人主页）。

## 6. 步骤五：调试与发布
1. **预览调试**：在右侧测试窗尝试提问：“请介绍一下黄运添在九号公司的实习经历？”或“你的联系方式是什么？”
2. **发布**：
   - 点击右上角“发布”。
   - 选择发布渠道：推荐发布至“微信客服”或生成一个“网页链接”直接放在您的电子签名中。
   - 勾选个人作品集展示。

## 7. 维护建议
- **定期更新**：每当有新的实习产出或获得新的技能认证时，请及时更新知识库中的文档。
- **日志分析**：在 Coze 后台通过“对话分析”查看面试官最关心哪些问题，持续优化 Prompt。

# README.md - AI Job Search Assistant Skill

AI 岗位求职助手技能 - 帮助用户进行全面的求职调研，包括搜索招聘信息、筛选匹配公司、深度调研企业背景、分析匹配度并提供简历优化和面试准备建议。

## 功能介绍

这是一个专为 AI 岗位求职者设计的 Claude Code 技能，提供系统性的求职调研和准备支持。技能分为 5 个步骤，从招聘信息搜索到最终报告生成，覆盖求职全流程。

### 核心功能

1. **招聘信息搜索**：多渠道搜索 50 家正在招聘目标岗位的公司

2. **简历解析与筛选**：解析简历，筛选出 20-25 家重点关注公司

3. **深度企业调研**：对前 15 家公司进行全面的背景调研（企业文化、工作制度、福利待遇等）

4. **匹配度分析与优化**：对前 8 家公司进行深度匹配分析，提供简历优化和面试准备建议

5. **完整报告生成**：整合所有分析结果，生成结构化的求职调研报告

## 适用岗位

- AI 产品经理

- AI 算法工程师

- AI 数据分析师

- AI 相关技术岗位

- 其他需要系统性求职调研的岗位

## 安装使用

### 方法 1：在 Claude Code 中直接安装

1. 在 Claude Code 中运行以下命令安装技能市场插件：

```bash
claude config plugins.marketplace.add https://github.com/MiaIria/yuntian.huang886
```

2. 然后选择 `Browse and install plugins`

3. 选择 `yuntian.huang886`

4. 选择 `ai-job-search-assistant-skill`

5. 点击 `Install now`

### 方法 2：本地安装

1. 克隆本仓库：

```bash
git clone https://github.com/MiaIria/yuntian.huang886.git
cd yuntian.huang886
```

2. 在 Claude Code 中加载本地技能：

```bash
# 在Claude Code中运行
/claude skills add ./ai-job-search-assistant-skill
```

### 方法 3：直接复制技能文件

1. 将 `SKILL.md` 文件复制到 Claude Code 的 skills 目录：

   - macOS: `~/.claude/skills/`

   - Linux: `~/.config/claude/skills/`

   - Windows: `%APPDATA%\Claude\skills\`

2. 创建技能文件夹并复制文件：

```bash
mkdir -p ~/.claude/skills/ai-job-search-assistant-skill
cp SKILL.md ~/.claude/skills/ai-job-search-assistant-skill/
```

## 使用方法

安装完成后，在 Claude Code 中使用以下方式调用技能：

### 基本用法

```plaintext
使用 ai-job-search-assistant-skill
目标岗位：AI产品经理
城市偏好：北京、上海
排除公司：腾讯、字节跳动
```

### 完整用法

```plaintext
请使用 ai-job-search-assistant-skill 帮我进行求职调研：

- 目标岗位：AI产品经理
- 城市偏好：北京优先，也可考虑上海和深圳
- 排除公司：腾讯、字节跳动、阿里巴巴、美团
- [上传简历PDF]
```

### 在 Claude Code 中直接调用

```bash
# 使用技能名称调用
/claude skills run ai-job-search-assistant-skill

# 或使用技能命令（如果配置了）
/ai-job-search-assistant-skill
```

## 输入要求

使用本技能时，请提供以下信息：

| 项目 | 说明 | 必填 |
| --- | --- | --- |
| 目标岗位 | 如“AI 产品经理”、“AI 算法工程师” | ✅ |
| 城市偏好 | 如“北京、上海优先”或“只考虑杭州” | ✅ |
| 个人简历 | PDF 格式的简历文档 | ✅ |
| 排除公司 | 已调研过或不考虑的公司列表 | ❌ |

## 输出内容

技能执行完成后，将生成一份完整的求职调研报告，包含：

1. **候选人画像总结** - 基于简历的个人优势和短板分析

2. **完整公司列表（50 家）** - 所有搜索到的招聘公司

3. **重点关注公司（20-25 家）** - 初步筛选后的高匹配度公司

4. **深度调研报告（15 家）** - 涵盖公司概况、企业文化、工作制度、福利待遇、AI 产品团队

5. **优先投递建议（8 家）** - 匹配度分析、简历优化建议、面试准备技巧

6. **通用求职建议** - 投递策略、面试技巧、注意事项

7. **附录** - 本次调研公司列表（供下次排除重复使用）

## 注意事项

1. **信息时效性**：招聘信息变化较快，建议在生成报告后 2 周内完成投递

2. **简历真实性**：所有简历优化建议基于真实经历，不建议编造或夸大

3. **客观中立**：企业调研基于公开信息，不同部门和时期可能存在差异

4. **隐私保护**：技能会保护用户隐私，不在报告中泄露联系方式
